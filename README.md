# SoberSpace Site

Static landing page for the SoberSpace app screenshots.

## Public Policy URLs

Use these URLs for app store metadata and in-app legal links once the site is deployed at `https://soberspace.app`:

```text
https://soberspace.app/privacy
https://soberspace.app/terms
https://soberspace.app/community-guidelines
https://soberspace.app/support
https://soberspace.app/account-deletion
```

## Start Locally

```bash
npm start
```

Then open:

```text
http://127.0.0.1:5173
```

If you do not want to use npm, run the same server command directly:

```bash
python3 -m http.server 5173 --bind 127.0.0.1
```
